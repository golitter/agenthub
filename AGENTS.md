Monorepo 项目，包含前端、后端、Agent 端三个子项目。

## 目录结构

```
bytedanceai/
├── frontend/      # React 前端 → 参见 frontend/AGENTS.md
├── backend/       # Go 后端   → 参见 backend/AGENTS.md
├── agentend/      # Python Agent 端 → 参见 agentend/AGENTS.md
├── docs/          # 项目文档
└── scripts/       # 工程脚本
```

子项目的框架选型、构建命令、测试方式等详情，请查阅对应目录下的 AGENTS.md。
