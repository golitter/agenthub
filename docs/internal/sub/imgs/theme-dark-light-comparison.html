关于本对比 —— 两栏渲染完全一致的 IM 群聊场景（图标栏 / 会话列表 / 消息流 / 输入栏 + 完整 token 参考表），仅 .dark class 取值不同，可直接对照明暗表现。
  • 主题机制：单一来源 index.css 定义 :root(Light) 与 .dark 双套 CSS 变量；use-theme.ts 通过 classList 增删 .dark 切换，默认 Dark。
  • 双主题共享：品牌色 #6366F1、3 个语义色、4 个 Agent 标识色在明暗下取值完全一致 —— 色彩只承担「功能」与「身份」，不随主题漂移。
  • 差异集中在：5 级背景灰阶、3 级文字灰阶、边框透明度（暗色 rgba(255,255,255,.06) / 亮色 #E5E5E5）、代码块底色（#0D0F14 / #F6F8FA）。
  • 身份表达：群聊中 Agent 通过 圆角方形头像辉光（box-shadow: 0 0 8px <color>）+ 类型徽标（彩色 mono 文字 + 同色 14% 底）区分，深度全部由背景色差表达，近乎不使用 box-shadow。
