实现 Markdown 风格增强

本文档说明如何在项目中实现 Markdown 排版增强，采用 @tailwindcss/typography 插件 + CSS 变量覆盖 + React components 自定义的三层策略。

实施步骤

1. 安装依赖

运行以下命令安装 Typography 插件：

pnpm add -D @tailwindcss/typography

2. 配置插件

在 src/index.css 顶部增加导入：

@import "tailwindcss";
@import "@tailwindcss/typography";  /* ← 新增 */
@import "tw-animate-css";
@import "shadcn/tailwind.css";

3. 覆盖 prose 颜色

在 .dark { } 块中添加 prose 语义变量，在 @layer components 中将其映射到 --tw-prose-*：

/* 暗色主题 prose 变量 */
--prose-heading: #F0F2F7;
--prose-link:    #818CF8;
--prose-code-text: #C7D2FE;

/* 映射到 Tailwind Typography 变量 */
.prose {
  --tw-prose-headings: var(--prose-heading);
  --tw-prose-links:    var(--prose-link);
  --tw-prose-code:     var(--prose-code-text);
}

4. 补全 React components

在 MarkdownRenderer.tsx 中补全 h1 ~ h4、blockquote、a、ul、ol、hr、strong、em 等元素的渲染组件。

提示：修改 components 对象后，热重载会自动生效，无需重启服务。

元素修改前修改后

注意事项

  • 修改仅涉及 3 个文件：package.json、index.css、MarkdownRenderer.tsx
  • 现有 CodeBlock.tsx（shiki 高亮）不受影响
  • fenceTreeBlocks 预处理逻辑不受影响
  • Tailwind CSS 4.x 使用 @import 加载插件，无需修改 vite.config.ts
完整修改方案详见 markdown-style-enhancement.md 修改报告。
